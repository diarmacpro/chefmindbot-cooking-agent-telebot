# chefmindbot-cooking-agent-telebot
ChefMindBot adalah AI Agent cerdas berbasis Telegram yang dirancang untuk mengakhiri drama "hari ini masak apa?". Layaknya asisten pribadi di dapur, bot ini memberikan ide masakan kreatif, daftar bahan belanja yang presisi, hingga panduan memasak langkah-demi-langkah layaknya Cookpad, namun dengan kecerdasan AI yang lebih personal.
